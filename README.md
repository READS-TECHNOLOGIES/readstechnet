# $READS Technologies — Marketing Site

**Live URL:** [readstechnet.vercel.app](https://readstechnet.vercel.app)

The official marketing and information site for **READS Technologies** — a Study-to-Earn crypto platform built on the Cardano blockchain.

---

## About $READS

$READS is a Learn-to-Earn educational platform that rewards Nigerian students (and learners globally) with `$READS` tokens for studying, completing quizzes, and passing exams like JAMB and WAEC.

- **Token:** $READS
- **Blockchain:** Cardano Native
- **Total Supply:** 100,000,000,000 (100 Billion)
- **App:** [reads-phi.vercel.app](https://reads-phi.vercel.app)
